# Obsidian: корень vault и превью глоссария

**Версия:** 1.0  
**Дата:** 2026-05-30

Документация продукта рассчитана на vault с **корнем репозитория `planner`** (весь монорепозиторий) или на подпапку **`obsidian-motivator`**. Ссылки на переменные и методы оформлены **одинаково** в обоих случаях.

## Ссылки на глоссарий (универсальный формат)

| Элемент | Формат |
|---------|--------|
| Переменная / метод | `[[dateKey#Определение\|dateKey]]` — **короткое имя файла** + заголовок **Определение** |
| Поле объекта / результата | `[[doubleConfirmPending#^glossary-def-confirmDeadlineIso\|confirmDeadlineIso]]` — страница-родитель + блок после callout |
| Индекс | [[00-Индекс]] |

Имена файлов в `18-Глоссарий/` **уникальны** во всём vault `planner`; Obsidian находит заметку по basename без префикса `obsidian-motivator/`.

**Не использовать** путь `[[dateKey#…]]` при vault = корень `planner` — ссылка **битая**, превью не появится.

## Page preview (подсказка при наведении)

1. **Корень vault:** папка **`planner`** (репозиторий) — конфиг `.obsidian` в корне репо; плагин **Page preview** включён.
2. **Режим редактирования:** в vault зафиксировано `obsidian-motivator/.obsidian/page-preview.json` → `"editor": false` — **превью по наведению в редакторе выключено**. Для подсказок по глоссарию переключитесь в **режим чтения** или включите Page preview для редактирования в **Настройки → Page preview**.
3. **Режим чтения:** обычное наведение на ссылку `[[идентификатор#Определение|…]]`.
4. На странице глоссария блок `^glossary-def` стоит **после** callout (отдельная строка); в тексте DR ссылаются на заголовок **`#Определение`** — так превью стабильнее, чем ссылка внутрь callout.

## Проверка

Откройте [[12-Журнал-решений#DR-001 — Стрик: что считается успешным днём]] и наведите на [[eodCompletedLocalDates#Определение|eodCompletedLocalDates]] (с Ctrl в режиме редактирования, если не отключали требование модификатора). Должен появиться callout «Определение».

## Сопровождение

- Починка якорей после callout: `node obsidian-motivator/18-Глоссарий/_scripts/fix-block-ids.mjs`
- Повторная миграция коротких ссылок: `node obsidian-motivator/18-Глоссарий/_scripts/migrate-universal-links.mjs`
