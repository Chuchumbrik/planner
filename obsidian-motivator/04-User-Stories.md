# User Stories (примеры для бэклога трекера)

Формулировки готовы для переноса в Jira/Trello; приоритизация — на стороне продукта.

## Двойное подтверждение

**Как** пользователь с СДВГ  
**хочу** при создании задачи сразу выбрать, будет ли двойное подтверждение и через сколько минут  
**чтобы** не забыть настроить это позже.

## End-of-day: включение и участие задач

**Как** пользователь  
**хочу** глобально включить end-of-day ритуал и для каждой задачи отметить, участвует ли она  
**чтобы** ритуал не раздувался на всё подряд.

## End-of-day: только выбранные задачи и вопросы

**Как** пользователь  
**хочу** в конце дня получить **3 выбранных мной** КПТ-вопроса **только по задачам**, отмеченным для ритуала  
**чтобы** рефлексия была релевантной и короткой.

## Анимации

**Как** пользователь  
**хочу** видеть приятную анимацию при подтверждении выполнения и мягкую — когда задача не выполнена  
**чтобы** эмоциональная обратная связь поддерживала, а не давила.

## Экспорт и контроль данных

**Как** пользователь  
**хочу** экспортировать **все данные** и **ключ шифрования** одним действием (или явным сценарием)  
**чтобы** иметь резерв и переносимость ([[06-Приватность-и-безопасность]]).
