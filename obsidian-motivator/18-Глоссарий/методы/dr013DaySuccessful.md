# dr013DaySuccessful

## Определение

> [!abstract] Определение
> Проверяет, засчитывается ли календарный день успешным для стрика: завершён EOD за [[dateKey#Определение|dateKey]] и (план пуст **или** есть ≥1 выполненная задача из плана).

^glossary-def

**Тип записи:** метод  
**Версия:** 1.0  
**Дата:** 2026-05-30

## Принимаемые типы данных

| Параметр | Тип | Описание |
|----------|-----|----------|
| `tasks` | `Task[]` | Список задач |
| `dateKey` | `string` | Ключ дня `YYYY-MM-DD` |
| `eodCompletedDates` | `Set<string>` | Множество дат с завершённым EOD |
| **Возвращает** | `boolean` | `true` — день успешен для стрика |
