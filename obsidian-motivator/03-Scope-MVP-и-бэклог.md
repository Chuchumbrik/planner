# Scope MVP и бэклог

**Актуальный пакет требований к MVP:** [[16-TZ-MVP-v1.0]] (**продукт v1.0.0**, редакция ТЗ **1.6**, 2026-05-09). Стратегическое решение зафиксировано в [[12-Журнал-решений#DR-012 — Стратегия MVP: календарь, планирование, приоритеты (MVP v1.0)]]. План внедрения по фазам: [[17-План-реализации-MVP]].

## Обязательно в MVP (сводка)

| Область | Содержание |
|---------|------------|
| Платформа | Web + PWA; целевое offline-capable — см. [[08-Архитектура]], [[17-План-реализации-MVP]] (фаза 9) |
| Локализация | RU + заготовка i18n (как минимум **en** в текущей реализации) |
| Сущности | Задача; группа / цвет; **приоритеты 1–5** (+ названия); **план работы** (чек-лист); **Backlog** |
| Планирование | **День** (локальная дата устройства), **неделя** (таймблокинг), **месяц** (календарь + фильтры); оценка времени; слоты; пересечения с предупреждением |
| Повторы | Каждый день; каждые N дней; по дням недели |
| Приоритеты | Полностью настраиваемая шкала **1–5** (не промежуточные «1–3 / Эйзенхауэр» как финальная модель) |
| Двойное подтверждение | По задаче; см. [[12-Журнал-решений#DR-004 — Двойное подтверждение пропуск второго шага]] |
| End-of-Day | Глобальный вкл/выкл; учёт по задаче; КПТ-банк — см. [[12-Журнал-решений#DR-002 — End-of-day какие задачи попадают в ритуал]] (уточнить под календарь при реализации) |
| Стрик | [[12-Журнал-решений#DR-013 — Стрик: уточнение под MVP v1.0 и DR-001]] |
| Анимации | Минимум **2 пары** тонов (успех / мягко при «не сделал») |
| Отчёты | На клиенте [[DR-007]]: доли, диаграммы, стрик, часто проваленные [[DR-008]] |
| Шифрование | Клиентское; ключ / seed — [[DR-005]], [[DR-006]] |
| UX обязательное | Просмотр vs редактирование; палитра; фильтры + информер; настройки по группам + смена пароля входа; удаление аккаунта + восстановление 30 дней по ключу (см. TZ §3–4) |
| Монетизация | Freemium — [[05-Freemium]], [[DR-010]] (реализация после ядра — [[17-План-реализации-MVP]] фаза 8) |
| Уведомления | Настраиваемые **PWA push** — в общем Scope продукта; в [[16-TZ-MVP-v1.0]] не детализированы — уточнять в спринте перед релизом |

## Не входит в MVP (v1.1+)

По [[16-TZ-MVP-v1.0]] §5 и расширению продукта:

- Глобальные цели с метрикой прогресса.
- Шаблоны задач + конструктор полей.
- Справки по long-press.
- Геймификация, AI и т.д.
- Пункты из раздела «Не входит» ниже (legacy PRD).

Дополнительно (из прежнего списка, не противореча ТЗ):

- AI-коуч, голосовой ввод, внешние интеграции Calendar/Telegram — по мере отдельных решений.
