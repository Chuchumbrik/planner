# Scope MVP и бэклог

## Обязательно в MVP

| Область | Содержание |
|---------|------------|
| Платформа | Web + PWA, offline-capable (целевое поведение — см. [[08-Архитектура]]) |
| Монетизация | Freemium ([[05-Freemium]]) |
| Локализация | RU в интерфейсе; заготовка i18n |
| Приоритеты | Выбор при старте: **1–3 уровня** или **матрица Эйзенхауэра**; смена в настройках |
| Задачи | Создание/редактирование, подзадачи, декомпозиция целей |
| Организация | Группировка по **цветам** и **проектам/группам** |
| Двойное подтверждение | Настройка **по задаче**; по умолчанию второй запрос **+10 мин** после первого |
| End-of-day | Глобальный вкл/выкл; учёт участия **по задаче**; **3 вопроса** по умолчанию; базовый набор **10 КПТ-вопросов** |
| Анимации | Минимум **2 пары**: успех/подбадривание и мягкая мотивация при «не сделал» |
| Уведомления | Настраиваемые **PWA push** |
| Отчёты | % выполнения день/неделя/месяц; столбчатые диаграммы (по дням; по дням недели за месяц; по группам); **стрик**; таблица **часто проваленных задач** |
| Профиль | Настройки + **wizard первого запуска** |

## Не входит в MVP (бэклог)

- AI-коуч, адаптивная аналитика.
- Геймификация с персонажем и XP.
- Голосовой ввод.
- Интеграции (Google Calendar, Telegram-бот и т.д.).
- «Верхнеуровневая премиум-аналитика» (rule-based) — позже.
- Пользовательские кастомные анимации.
