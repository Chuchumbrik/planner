# Технический стек (рекомендация PRD)

Ориентация: **бесплатный старт**, минимальный vendor lock-in, PWA и offline.

## Frontend

| Компонент | Выбор |
|-----------|--------|
| Фреймворк | React **19** |
| Язык | TypeScript |
| Сборка | Vite |
| Стили | Tailwind CSS |
| UI-кит | shadcn/ui |
| Серверное состояние / API | TanStack Query |
| Клиентское состояние | Zustand |
| PWA | Vite PWA plugin |
| i18n | react-i18next (заготовка + английский как второй язык структуры) |

## Визуализация и motion

- Диаграммы: **Recharts** или **Chart.js**
- Анимации: **Framer Motion**

## Backend / инфраструктура

- **Supabase**: Auth, Postgres, Storage; Realtime — по необходимости.
- В перспективе: **self-host** Supabase / Postgres без смены модели шифрования на клиенте.

## Хостинг (старт)

- Frontend: **Vercel**
- Backend: **Supabase** (managed)

## Шифрование

- **Web Crypto API** в браузере (без сторонних проприетарных KMS на MVP).
