## Этап 7. Адаптив и PWA — формальный аудит (1–2 дня) ✅ 2026-05-24

Чеклист по экранам (`/`, `/login`, `/onboarding`, `/app` ×3 вкладки, `/app/reports`, `/settings`):

- [x] Нет horizontal scroll на 320–430px
- [x] Tap targets ≥ 44×44px (кнопки, чекбоксы, табы)
- [x] `env(safe-area-inset-*)` — header, bottom nav, modals
- [x] PWA: `theme-color` #131315, standalone, viewport
- [x] Breakpoints: `sm` / `md` / `lg` / `xl` — предсказуемые скачки, не «ломается между»
- [x] Keyboard: focus ring primary на инпутах
- [x] Reduced motion: не блокер, но без лишних бесконечных pulse
